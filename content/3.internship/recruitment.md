## KEEHIN Software Development

- **ระยะเวลา:** 2–4 เดือน
- **ตำแหน่ง:** นักศึกษาฝึกงานสาย **IT / Data / Developer**

## 🔍 ลักษณะงานที่รับผิดชอบ

🗄️ **Data / Database**

- 🔎 วิเคราะห์และดึงข้อมูลด้วย SQL
- 🛢️ ใช้งาน **MariaDB** และ **SkySQL**
- 🖥️ ใช้งานฐานข้อมูลผ่าน **DBeaver**

🧑‍💻 **Development / Web Application**

- 🌐 พัฒนาเว็บแอปด้วย **Nuxt.js**, **Node.js**, **TypeScript**
- ใช้งาน **GitHub** เพื่อจัดเก็บและเผยแพร่โค้ด

📊 **Report & Visualization**

- 📝 ออกแบบรายงานด้วย **JasperReport Studio**
- 📈 ทำ Visualization ด้วย **Nuxt.js**

🖥️ **Server System (Linux)**

- 🐧 ใช้งานเซิร์ฟเวอร์ **Linux (Debian / Ubuntu)**
- ⚙️ จัดการซอฟต์แวร์ Backend เช่น **Java, Node.js**

🛠️ **IT Support**

- 🖨️ ดูแล ซ่อมแซม และตรวจสอบอุปกรณ์ IT
- 💾 ติดตั้งซอฟต์แวร์พื้นฐานให้ผู้ใช้งานภายในองค์กร

## 🎯 Project-Based Learning (โครงการฝึกงาน)

- เรียนรู้และพัฒนา จากการสร้างโปรเจคซอฟแวร์ ขึ้นอยู่กับทักษะพื้นฐาน และความถนัดของผู้ฝึกงาน

## 🧑‍🎓 คุณสมบัติผู้สมัคร

- มัธยมปลาย / ปวส. / ป.ตรี สาขาคอมพิวเตอร์ หรือที่เกี่ยวข้อง
- มีพื้นฐานการเขียนโปรแกรม ไล่ลำดับการทำงาน แก้ไขจุดผิดพลาดได้
- รับผิดชอบหน้าที่ และชอบแก้ไขปัญหา
- ผ่านแบบทดสอบ ความเข้าใจ [ตัวอย่างโปรแกรม example.html](/code/example.html){target="_blank"}
