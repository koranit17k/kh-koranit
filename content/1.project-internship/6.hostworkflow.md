## ขั้นตอนการทำ host ของเว็บ kh-koranit และ การเรียกใช้งาน report viewer

*จำเป็นต้องมี project kxreport และ kh-group ในเครื่อง
- ลงโปรแกรม nginx(เปรียบเสมือนเครื่องเซิฟเวอร์) ที่เครื่องที่จะทำ host ผ่าน sudo apt install nginx
- หลังจากนั้น ให้เข้า project และ pnpm build เพื่อ ลง nginx ใน project เพื่อจะได้ไฟล์ dist ใน project kh-koranit   
- หลังจากนั้น ให้ copy ไฟล์ dist ไปไว้ที่ /var/www/dist
    ```bash 
    sudo mkdir -p /var/www/dist
    ```
- หลังจากนั้นตั้ง permission ให้ไฟล์เพื่ออนุญาต การเข้าถึง เพื่อที่คนอื่นจะได้ใช้ ip ของเครื่องมาใช้ได้
    
    *อันตราย หาก ip หลุดไป*

    ```bash sudo chmod 755 /home/koranit ```

    ```bash sudo chmod 755 /home/koranit/attendance```

    ```bash sudo chmod 755 /home/koranit/attendance/index.html```

    ```bash sudo chmod 755 *.jasper```

    ```bash cp /home/koranit/attendance/report/*.jasper /khgroup/report/payroll  <!--copy อีกครั้งนึง-->```

    ```bash sudo cp -r /home/koranit/kh-koranit/dist/* /var/www/dist    <!--copy ไฟล์ให้เข้าไปใน nginx-->```

- ไปแก้ ไฟล์ config ของ nginx  เพื่อตั้งค่า nginx รับรู้ว่ามีเว็บอะไรบ้าง ตั้งเว็บไหน เป็น host หลัก และอะไรเป็นเว็บย่อย     ผ่าน 
```bash
sudo nano /etc/nginx/sites-available/default
```

```bash
server {
        root /var/www/dist; <!-- kh-koranit host เว็บหลัก-->
        charset utf-8;
#       location / {
#               try_files $uri $uri/ =404;
#       }
<!--/att เป็น kxreport viewer-->
    location /att {         
        alias /home/koranit/attendance;
        index index.html;
#        try_files $uri $uri/ /index.html;
    }
}
```

- หลังจากนั้นก็สามารถเข้าผ่าน
เช็คผ่าน command line
```bash
ip address
```
http://192.168.30.162/ จะเป็น kh-koranit host หลัก
http://192.168.30.162/att/ เป็น kxreport viewver
การจะเปิดผ่าน host ของเครื่องหลัก จำเป็นต้องเปิดเครื่อง และเช็ค ip เครื่องก่อน
