## Vitest program for test scenarios
- ติดตั้ง vitest https://vitest.dev/guide/
    - ติดตั้ง ผ่าน command line 
    ```bash
    pnpm add -D vitest
    ```
- สร้าง folder test และ ไฟล์ .test
- ทดลอง สร้างไฟล์ sum.js
 ```bash 
export function sum(a, b) {
  return a + b
}
``` 
- ไฟล์ sum.test.js
 ```bash 
import { expect, test } from 'vitest'
import { sum } from './sum.js'

test('adds 1 + 2 to equal 3', () => {
  expect(sum(1, 2)).toBe(3)
})
 ```

- รัน test case
     ```bash 
        pnpm test
     ```

- หลังจากทดลองเทส แล้วผ่านให้ทำเทสของ attendance ต่อ
- สร้างไฟล์ att.test และให้ agent ai ช่วยสร้างเทสให้ผ่าน test case screnarios ที่เคยทำไว้ใน @exampledata โดยการคำนวนของ myview.sql (vAttendance)
- หลังจากให้ ai generate test case แล้ว ให้ลองรัน test case โดยใช้คำสั่ง pnpm test att และดูผลลัพธ์
- หากไม่ผ่านให้กลับไปแก้ procedure หรือ myview(query ที่คำนวนเวลา) และรัน test case ใหม่ โดยใดยสามารถใช้ทั้ง ai และ เช็คด้วยมือ (อาจจะมีคำนวนผิดพลาดที่ test caseจึงควรเช้คให้ละเอียด )
- ให้ check test ผ่าน pnpm test att จนผ่าน 100%
