## Vitest program for test scenarios
- หลังจาก[ติดตั้ง vitest](../1.project-internship/installation#9.vitest)
- สร้าง folder test และ ไฟล์ .test ใน project
- ทดลอง สร้างไฟล์ sum.js
 ```bash 
export function sum(a, b) {
  return a + b
}
``` 
- ไฟล์ sum.test.js
 ```bash 
import { expect, test } from 'vitest'
import { sum } from './sum.js'

test('adds 1 + 2 to equal 3', () => {
  expect(sum(1, 2)).toBe(3)
})
 ```

- รัน test case ด้วยคำสั่ง
     ```bash 
        pnpm test
     ```

- หลังจากทดลองทดสอบแล้วและผลผ่าน ให้ดำเนินการทดสอบในส่วนของ Attendance ต่อ
- สร้างไฟล์ `att.test` และใช้ Agent AI ช่วยสร้างชุดทดสอบให้ครอบคลุมตาม test case scenarios ที่เคยจัดทำไว้ใน `@exampledata` โดยอ้างอิงการคำนวณจาก `myview.sql` (`vAttendance`)
- หลังจากให้ AI Generate Test Case แล้ว ให้ทดลองรันชุดทดสอบด้วยคำสั่ง `pnpm test att` และตรวจสอบผลลัพธ์
- หากผลการทดสอบไม่ผ่าน ให้กลับไปแก้ไข `procedure` หรือ `myview` (Query ที่ใช้คำนวณเวลา) แล้วจึงรันชุดทดสอบใหม่ โดยสามารถใช้ทั้ง AI และการตรวจสอบด้วยตนเองร่วมกันได้ เนื่องจากอาจมีความผิดพลาดทั้งในส่วนของการคำนวณและใน Test Case จึงควรตรวจสอบอย่างละเอียด
- ตรวจสอบและรัน `pnpm test att` จนกว่าผลการทดสอบจะผ่าน 100%
