## ทดสอบการใช้งานระบบการเรียกใช้ report

1. Clone หรือ Fork Repository จาก https://github.com/swasin185/kxreport.git  
   - *ต้องติดตั้ง Node.js, pnpm และ Java ก่อน*
2. หลังจากสร้างรายงานจากไฟล์ `.jrxml` แล้ว ให้ Compile ไฟล์ `.jrxml` เป็นไฟล์ `.jasper`
3. รัน Script `install.sh` เพื่อติดตั้งโปรแกรมที่ใช้สำหรับแสดง Report
4. รัน Script `build.sh` เพื่อสร้างโครงสร้างระบบ
5. ทดสอบการเรียกใช้งานผ่าน API ที่ `http://localhost:8080/kxreport/`
6. รัน Script `mem.sh` หรือ `stress.sh` เพื่อทดสอบการใช้ RAM
7. รัน Script `copyreport.sh` เพื่อคัดลอกไฟล์ `.jasper` ไปยังโฟลเดอร์ `payroll`
8. ทดลองใช้ระบบผ่าน api http://localhost:8080/kxreport/getPDF?report=A02&db=payroll&startDate=2025-12-01&endDate=2025-12-31&&comCode=01


## Note*
- ต้องมี kh-group ใน root และต้องมี project kxreport
- ไฟล์ที่จะให้ report จะแสดงในเว็บอยู่ใน path:/khgroup/report/payroll/
- สามารถทดสอบระบบ จากข้อที่ 5 โดนผ่าน parameter ในเว็บได้เลย ผ่าน url ตัวอย่างเช่น
http://localhost:8080/kxreport/getPDF?report=A02&db=payroll&startDate=2025-12-01&endDate=2025-12-31&&comCode=01
