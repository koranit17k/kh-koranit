## config nginx สำหรับรัน local
ไปหาไฟล์
```bash
cd /etc/nginx/sites-enabled
```

เข้าแก้
```bash
sudo nano default
```

แก้ config ของ nginx เพื่อรัน local เป็น
```bash
server {
        charset utf-8;
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /att {
        alias /home/koranit/attendance;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
}
```
หลังจากนั้น ใช้คำสั่งในไฟล์ 3.npxdeploy.sh เพื่อ deploy งาน
